BeatsByUzo

Wordpress / PHP / JQuery / HTML / Sass

In my free time I produce and compose hip-hop/R&B instrumentals. I wanted a way to combine the features of a music player with the functionality of a digital goods store, so I collaborated with my friend Steve Fraser and created one.

Wordpress uses a plugin system that allows you to inject pieces of code where you need to, which lets it act as a glue for all of the different components.

  • PHP is used for all the complex back-end data processing.
  • JSON was used to transfer form data back and forth via AJAX. This included PayPal checkout, form submissions, etc.
  • Sass/Compass was used to write the CSS, especially for the color effects on the player
  • MySQL was used to enter the songs into database forms and keep track of ratings, prices, file delivery URLs, etc.
  • Date: July 2013
  • Languages Used: PHP / JQuery / HTML / Sass / MySQL / JSON / AJAX / Paypal API
